Проект «ТЕТРИС»
Автор: Автухович Наталья

Мой проект представляет собой упрощенный вариант игры тетрис.
Цель игрока: набрать необходимое количество очков и перейти на следующий уровень или остаться на текущем.
Очки начисляются за каждый собранный нижний ряд. Фигуры можно двигать влево-вправо-вниз при помощи стрелочек,
а также поворачивать их при помощи стрелки вверх (только со 2-го уровня).

Вся программа реализована с использованием спрайтов. Каждая фигура - даже вспомогательная -
представляет собой спрайт. Большое количество функций помогает быстро разобраться в коде и найти/ исправить
нужный фрагмент.

Интересна реализация поля. Она храниться в отдельном списке field, который помогает контролировать перемещение спрацтов.

Основным классом является класс Figures, который игнтересен тем,
что создаваемая фигура представляет собой группу спрайтов класса Alone,
которые удаляются из фигуры при сборке нижней строки.

В проекте я использовала:
import sys - для запуска игры
import csv - для создания и записи результатов
import pygame - сам pygame
from random import choice - рандомная генерация цветов

библиотека pyQt5 для отображения результатов
from PyQt5.QtWidgets import QApplication, QWidget, QTableWidgetItem
from PyQt5 import QtCore, QtGui, QtWidgets
