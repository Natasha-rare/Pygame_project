Индивидуальный проект "Тетрис"

В данном проекте будет осуществлена игра тетрис с нескольким количеством уровней (уровни разделены по скорости).
Пользователю будет дана возможность выбрать уровень, а так же остановить игру и начать уровень заново.
Будут реализованы очки (за правильное прохождение уровней). Очками считается количество собранных рядов.

На заднем плане будет играть спокойная музыка.
В таблице рекордов будут хранится пройденные уровни (с возможностью введения игрока).
